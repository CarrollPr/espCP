Инструкция по переводу в режим прошивки:

ESP32 DevKit: Зажать кнопку BOOT (GPIO0), нажать RST, отпустить RST, затем отпустить BOOT.

ESP32-C3: Зажать кнопку BOOT при подключении или сбросе.

ESP8266: Удерживайте GPIO0 на земле при сбросе.

Если автоматический режим не сработал, попробуйте ручной метод для вашей платы.
